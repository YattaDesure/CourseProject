# Настройка TimeWeb MCP Server для Cursor

## Что это?

MCP (Model Context Protocol) сервер для интеграции Cursor с TimeWeb Cloud, позволяющий управлять деплоем прямо из редактора.

## Настройка

1. **Получите API токен TimeWeb:**
   - Зайдите в панель управления TimeWeb
   - Перейдите в раздел "API" или "Настройки разработчика"
   - Создайте новый API токен
   - Скопируйте токен

2. **Обновите конфигурацию:**
   - Откройте файл `.cursor/mcp.json`
   - Замените `"your-api-token"` на ваш реальный токен

3. **Перезапустите Cursor:**
   - Закройте и откройте Cursor заново
   - MCP сервер должен автоматически подключиться

## Использование

После настройки вы сможете:
- Управлять деплоем через Cursor
- Просматривать логи сборки
- Управлять контейнерами
- И многое другое

## Проверка

После перезапуска Cursor проверьте:
- В настройках Cursor должен появиться MCP сервер `timeweb-mcp-server`
- Статус должен быть "Connected"

## Проблемы?

Если MCP сервер не подключается:
1. Убедитесь, что токен правильный
2. Проверьте, что `npx` доступен в системе
3. Проверьте логи Cursor на наличие ошибок
